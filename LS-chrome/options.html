如何使用

  1. 先连上你的 VPN 或代理，这决定平台看到的真实 IP 属地。
  2. 在本页填写目标城市的纬度、经度、时区和语言，再把目标站点域名加入白名单。
  3. 点“保存设置”，如果有新增域名，浏览器会弹出授权窗口。
  4. 打开目标站点，点扩展 popup 里的“对当前站点启用”，𑛖后刷新当前页面。
  5. 最后回本页点“执行网络校验”，检查公网 IP 的时区是否与你的档案一致。
